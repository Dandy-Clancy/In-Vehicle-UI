# MySubaru Prototype — Screen Index
# Use with Figma plugin "html.to.design" or screenshot each URL

## How to import into Figma:

### Option 1: html.to.design plugin (RECOMMENDED)
1. Install "html.to.design" plugin from Figma Community
2. Open each URL below in the plugin
3. It converts the HTML page into editable Figma frames

### Option 2: Screenshot tool
1. Open each HTML file in Chrome at 402x874
2. Use Chrome DevTools → Device Toolbar → set to 402x874
3. Screenshot each screen
4. Import screenshots into Figma

### Option 3: Figma "Import from URL" (if available)
Paste each file:// URL into the importer.
